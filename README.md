## Deployment

    -Open terminal
    -Go to your project (cd project-api/)
    -vagrant up
    -vagrant ssh
    -cd ../../vagrant
    -bundle install
    -rails db:setup
    -rails db:migrate
    -rails server
    -Open browser with the url generated.
